Lane Calibration

This page shows and updates the queue_geometry section of config.yaml. For best accuracy, first run the desktop tool tools/calibrate_queue.py and then check the values here.

Geometry values

After saving, restart run_camera_bt.py. When use_slanted_line: true, the system will use these values.

Important: Slanted-line calibration is meant for your 4K lane camera. With a laptop webcam, keep use_slanted_line: false and use the vertical band mode instead.

Desktop tool (recommended)

1. Run: python tools/calibrate_queue.py
2. Left click near the cashier (NEAR point).
3. Right click at the far end of the line (FAR point).
4. Press s to save.
5. Close the window and restart the lane runner.
